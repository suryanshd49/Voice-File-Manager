# 🎤 Voice File Manager

A Java-based desktop application that allows users to perform file operations using voice commands.

## 🚀 Features
- Create files using voice
- Delete files
- List files
- Voice recognition using Vosk API
- User-friendly GUI (Swing)

## 🛠️ Tech Stack
- Java
- Java Swing
- Vosk API
- Java Sound API

## 🎤 Example Commands
- "create file demo"
- "delete file demo"
- "list files"

## 📂 How to Run
1. Clone the repo
2. Add Vosk model
3. Run App.java
